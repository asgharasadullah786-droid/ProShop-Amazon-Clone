# 🛒 ProShop Amazon Clone (Laravel E-Commerce Project)

This is a full-featured **E-commerce web application** built using **Laravel**.  
It simulates an Amazon-like shopping system with user, admin, and product management features.

---

## 🚀 Features

### 👤 User Side
- User authentication (Login/Register)
- Product browsing and search
- Add to cart functionality
- Wishlist system
- Checkout process
- Order tracking
- Product reviews

### 🛠️ Admin Panel
- Product management (CRUD)
- Category management
- Order management
- User management
- Flash sales & sliders
- Dashboard analytics

---

## 🧑‍💻 Tech Stack

- Laravel (PHP Framework)
- MySQL Database
- HTML, CSS, JavaScript
- Bootstrap / Tailwind CSS
- Blade Templates
- Vite

---

## 📦 Installation

```bash
git clone https://github.com/asgharasadullah786-droid/ProShop-Amazon-Clone.git
cd ProShop-Amazon-Clone
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
